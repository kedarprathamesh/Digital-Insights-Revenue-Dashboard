# **Digital Insights & Revenue Dashboard**

This project presents an Excel-based analytics dashboard built using a Web Analytics Dataset. It highlights traffic patterns, user behavior, and revenue trends through dynamic visuals, pivot tables, slicers, and YOY metrics. The dashboard is designed for clear decision-making and performance tracking.
