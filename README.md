# **Digital Insights & Revenue Dashboard**

This project delivers a full Excel-based analytical dashboard built using a Web Analytics Dataset. It highlights traffic trends, user engagement, conversion behavior, and revenue performance through dynamic visuals, pivot tables, slicers, and clear YOY insights. Designed for data-driven decision-making and reporting.
